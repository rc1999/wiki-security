# Ripple Shares DPRK Threat Intelligence with Crypto Industry

**Analysis and Actionable Summary — May 2026**

---

## TL;DR

Ripple has begun sharing its internal North Korean (DPRK/Lazarus Group) threat intelligence with the crypto industry via **Crypto ISAC's new API**. The data includes DPRK-linked wallet addresses, malicious domains, and **enriched operative profiles** (LinkedIn IDs, emails, phone numbers, locations) to help firms detect insider-driven infiltration before catastrophic losses.

**Key numbers:**
- $577M lost to Lazarus in early 2026 (Drift $285M + Kelp DAO $292M)
- 76% of all crypto theft in 2026 attributed to North Korea
- >$6.7 billion stolen by DPRK since 2017

**Coinbase is already integrated.** The API normalizes Web2 + Web3 threat indicators for direct integration into security operations.

---

## The Threat: Social Engineering Infiltration

North Korean hackers have evolved from remote exploits to **long-term insider infiltration**:

1. Apply for jobs at crypto firms using fake identities
2. Build trust over months — attend conferences, meet in person
3. Compromise devices through malicious software
4. Bypass traditional IOC-based detection (no malware signatures)
5. Extract funds via compromised multisig wallets

### The Reapplication Problem

> "Threat actors rejected by one firm often reapply to several others the same week."

Without shared intelligence, each company screens applicants independently. A rejected operative at Company A applies to B, C, D the same week — each starting from zero. Crypto ISAC was built to close this gap.

## What Gets Shared via Crypto ISAC API

| Data Type | What's Included | How to Use It |
|-----------|---------------|-------------|
| **DPRK-linked wallets** | Blockchain addresses tied to Lazarus | On-chain monitoring, transaction screening, withdrawal blocking |
| **Malicious domains** | Fraud websites used in campaigns | DNS blocking, email filtering, phishing detection |
| **IOCs** | Technical signatures from active campaigns | SIEM integration, endpoint detection |
| **Enriched profiles** | LinkedIn, email, phone, location, correlated campaign signals | HR onboarding screening, background checks, contractor vetting |
| **Campaign correlation** | Connections between individuals and broader operations | Pattern analysis, network mapping |

**What makes it different:** A profile isn't just a name — it's a full identity package with correlated campaign signals. Transforms raw data into actionable intelligence.

## The Incidents That Triggered This

### Drift Protocol — $285M (Early 2026)
- **Platform:** Solana perpetual futures DEX
- **Method:** 6-month social engineering campaign
- **Tactics:** Gained trust of contributors, in-person meetings, malicious software, compromised multisig
- **Lesson:** Not a smart contract exploit — an inside job via trust manipulation

### Kelp DAO — $292M (Early 2026)
- **Platform:** Ethereum re-staking platform
- **Method:** Similar insider infiltration
- **Lesson:** Same playbook, different target

**Combined:** $577M = 76% of all crypto hack value in 2026 (through April)
