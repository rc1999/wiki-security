# Coinbase

**Major US cryptocurrency exchange. Founding member of Crypto ISAC and early adopter of its enhanced threat intelligence API.**

## Profile

| Attribute | Detail |
|-----------|--------|
| **Type** | Cryptocurrency exchange |
| **Founded** | 2012 |
| **Security lead** | Jeff Lunglhofer, Chief Information Security Officer |
| **Crypto ISAC role** | Founding member, API co-developer |

## Crypto ISAC Contribution

Coinbase helped shape the data model for Crypto ISAC's updated API:

> "Working with Crypto ISAC on developing their updated API allowed us to help shape a data model that preserves context and confidence — not just indicators — and supports both Web2 and Web3 use cases. As an early adopter, we've already seen how this improves our ability to act on intelligence in real time."
> — Jeff Lunglhofer, Chief Information Security Officer, Coinbase
