# Crypto ISAC

**The Crypto Information Sharing and Analysis Center — a member-driven, not-for-profit organization that enables coordinated threat intelligence sharing across the cryptocurrency ecosystem.**

## What It Does

- Curates and distributes threat intelligence among member crypto firms
- Develops APIs for normalized, contextually rich data sharing
- Bridges Web2 and Web3 threat indicators into unified formats
- Facilitates collective defense against nation-state and criminal actors
- Enables real-time intelligence sharing when threats are detected

## Key Technology: The Crypto ISAC API

Launched May 2026 — designed specifically for crypto defense:

| Feature | Description |
|---------|-------------|
| **Contextual enrichment** | Not just raw indicators — includes LinkedIn, email, location, campaign correlation |
| **High-confidence data** | Validated through AI-enhanced detection + human analyst review |
| **Web2 + Web3 normalization** | Unified format for traditional and blockchain indicators |
| **Direct integration** | Built for SIEM, SOAR, and onboarding system integration |
| **Real-time updates** | New intelligence flows to all members immediately |

## The "Reapplication Problem"

Crypto ISAC addresses a critical gap in crypto security:

> "A threat actor might fail a background check at one company and apply to three others the same week. Without shared intelligence, each company is starting from zero."

When one member detects a sophisticated threat actor, enriched profile data flows to every other member. If that actor applies elsewhere, those companies are already armed with real-time data.

## Philosophy

- **Actionability as the core function** — Intelligence only creates value when it can change an outcome
- **Collective defense as gold standard** — Information sharing is no longer optional
- **Context over raw signals** — Enriched data (who, where, how connected) beats isolated IOCs
